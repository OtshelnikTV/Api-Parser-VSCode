# API Parser

YAML API documentation parser and viewer for Visual Studio Code.

## Features

- Parse and view YAML API documentation
- Built-in web interface for API exploration
- Save documentation files directly from VS Code
- Open in external browser support

## Usage

1. Open the API Parser panel from the Activity Bar
2. Load your YAML API documentation
3. View and edit in the integrated interface

## Installation

Install from the VS Code Marketplace or download the VSIX file.

## Requirements

- VS Code 1.74.0 or higher
